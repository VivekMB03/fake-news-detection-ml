# Fake News Detection using Machine Learning

A Machine Learning based Fake News Detection system developed using Python.  
This project analyzes news articles and predicts whether the news is **Real** or **Fake** using Natural Language Processing (NLP) and Machine Learning techniques.

---

## Features

- Detects fake and real news articles
- Uses Machine Learning algorithms
- Text preprocessing using NLP techniques
- CSV dataset support
- Simple and easy-to-understand implementation
- Jupyter Notebook based project

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- NLP
- Machine Learning
- Jupyter Notebook

---

## Dataset

This project uses two datasets:

- `Fake.csv`
- `True.csv`

These datasets contain fake and real news articles used for training and testing the model.
